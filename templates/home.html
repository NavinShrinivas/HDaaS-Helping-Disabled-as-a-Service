Sub-Service of the service and how to use them:

Sub-Service : Tippoff-Avoider

  • This service is to be used to detect if a slope is safe to climb over

  • service call format : http://locahost:5000/tippoffservice/x/y/typ

  • OPTIONS :

    • x : height between flat floor and sensor [can be hard coded]
    • y : data recived from sensor of distance between sensor and slope
    • type :
      1. wa : without assistance saftey tolerances of ADA standards will be taken into account
      2. a : assisted saftey tolerances of ADA standards will be taken into account

  • OUTPUT : use web scrapper and split and the first index of the list will be either "False" or "True"

Sub-Service : TxtToBraille

  • this service covertes any text given to it into formas of braille in 1's and 0's
    we aimed on feeding this data into a mechnical braille and make a "dynamic braille"

  • service call format : http://localhost/txt-braille/y

  • OPTIONS :

    • y : can be any text with or without spaces

  • OUTPUT : on web scrappoing a list of 6 length string are given out wherer each index of str corresponds to a braille
